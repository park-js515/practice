# Jupyter Notebook Setting

> 환경 설정

1. 먼저 터미널에 다음과 같이 가상환경을 설정합니다.
```sh
python --version # Python 3.9.13
python -m venv venv
source venv/Script/activate
pip install -r requirements.txt
```

2. `ctrl + shift + p`를 누르거나 `설정-명령 팔레트`를 통해서 다음 설정을 찾습니다.
![select_interpreter](src/img/img1.png)

3. 우리가 만든 `venv`로 환경을 설정합니다.

4. 추가적으로 필요한 패키지의 경우 터미널에서 `pip install`을 활용해 설치하고 import 합니다.
